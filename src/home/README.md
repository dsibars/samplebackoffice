# Home Module

This is the initial module for the Sample Backoffice. It demonstrates the structural conventions for a standard module in this app.

## Directory Structure
- `domain/`: Business entities and interfaces (`HomeModel.ts`).
- `application/`: Application logic orchestration (`HomeService.ts`).
- `infrastructure/`: Data retrieval logic (`HomeStorage.ts`).
- `presentation/`: React components and UI code (`HomeView.tsx`).

When adding a new module in the future, replicate this structure to ensure the architecture remains clean.
